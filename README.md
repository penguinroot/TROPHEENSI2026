# 🌿 FAUNEX

Projet présenté pour les **#Trophées NSI 2026**

FAUNEX est un jeu 3D de photographie animalière développé en **Python**
avec **Ursina Engine**.\
Incarnez un photographe, explorez la nature, capturez les meilleures
images et enrichissez votre encyclopédie !

------------------------------------------------------------------------

## ✨ Fonctionnalités

### 📸 Système de Photo

-   Viseur interactif\
-   Calcul du score selon le cadrage et la distance\
-   Utilisation du **Raycasting** pour la précision des clichés

### 🦊 Écosystème Vivant

-   21 animaux différents\
-   IA comportementale :
    -   Fuite\
    -   Curiosité\
    -   Sommeil

### 🧠 Quiz & Wiki-Dex

-   Identification des espèces photographiées\
-   Gain de crédits\
-   Encyclopédie interactive à compléter

### ♻️ Gestion & Écologie

-   Ramassage des déchets\
-   Achat de cartes SD\
-   Achat d'appâts\
-   Gestion des crédits

### 💾 Sauvegarde JSON

-   Progression entièrement persistante\
-   Sérialisation et désérialisation des données

------------------------------------------------------------------------

## 🚀 Installation & Lancement

``` bash
git clone https://github.com/penguinroot/TROPHEENSI2026
cd TROPHEENSI2026
pip install ursina
python main.py
```
